# EC25B1067_MaRS_Task1
EC25B1067_SIDDHARTH S_MaRS_Task#1_Recruitment
The process uses LEDs and resistors and arduino uno to blink LEDs at intervals of 500 ms, 1000 ms, 1500 ms respcetively.
For this, millis() function was used to keep track of time and blink LEDs accordingly. Once an LED hits its interval, it switches its output to the opposite of the present value ( 1 to 0 or 0 to 1).
